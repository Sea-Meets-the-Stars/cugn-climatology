# cugn-climatology
Climatology for CUGN

Generates climatologies for the California Underwater Glider Network
([CUGN](https://spraydata.ucsd.edu/projects/CUGN/)).

## Installation

```bash
pip install -r requirements.txt
pip install -e .
```

This package builds on the `cugn` package, which is not on PyPI; install it
from a local checkout alongside this repository:

```bash
pip install -e ../cugn
```
