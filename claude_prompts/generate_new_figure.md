This prompt doc will generate a new climatology figure by only providing you a text description of the figure.  We will use Python code to generate the figure.

## Code

Follow these guidelines:

- Use inline comments to explain the code.
- Use matplotlib
- Code in Python
- Place imports at the top of modules
- Reuse any code in this repository to help you
- Use functions and not classes
- Include a description of inputs/outputs in the doc string of all methods 
- Add "Generated by JXP and Claude" in the doc string of all methods you create

If you need to run Python code, please use the `ocean14` conda environment.
